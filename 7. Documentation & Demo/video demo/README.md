# 🎥 Project Video Demonstration & Scenario Walkthrough

This folder contains the complete video demonstration and walkthrough of the **EcoSense – Sustainable Waste Management Assistant Using Generative AI** project. The demonstration showcases the application's key features through real-world scenarios, highlighting how AI assists users in identifying, classifying, and responsibly disposing of different types of waste.

## 🎬 Demonstration Scenarios

### 🔴 Scenario 1: Hazardous Waste Scanning (Battery)

**Description:**  
The user opens the EcoSense Scanner and searches for **"battery"** to analyze its disposal requirements.

**Features Demonstrated**

- 🤖 AI-powered waste classification using **Llama 3.3**
- 🚨 Hazard detection with **High Risk** classification
- ⚠️ Safety alerts for hazardous materials (Mercury, Lead, Cadmium)
- ♻️ Proper disposal instructions for safe handling
- 🌱 Eco-friendly recommendations such as using rechargeable batteries

---

### 🟢 Scenario 2: Recyclable Waste Sorting (Plastic Bottle)

**Description:**  
The user scans a **plastic bottle** to verify whether it is recyclable.

**Features Demonstrated**

- 🤖 AI identifies the item as **Plastic Waste**
- ♻️ Recyclability status displayed instantly
- 🧼 Waste preparation guidelines:
  - Empty remaining liquid
  - Rinse thoroughly
  - Remove the cap
  - Crush the bottle before disposal
- 📝 Automatic scan history logging for future reference

---

### 📍 Scenario 3: Nearby Recycling & Drop-off Center Locator

**Description:**  
After identifying hazardous waste, the user navigates to the interactive map to locate the nearest disposal center.

**Features Demonstrated**

- 🗺️ Interactive map powered by **Leaflet.js**
- 📍 Color-coded location markers:
  - 🔴 Hazardous & E-waste Centers
  - 🟢 Recycling Centers
  - 🔵 Organic Compost Centers
- 📋 Detailed location information including:
  - Center name
  - Address
  - Operating hours
  - Accepted waste materials
  - Contact details

---

### 📊 Scenario 4: Sustainability Analytics Dashboard

**Description:**  
The user views their recycling statistics and sustainability impact through the analytics dashboard.

**Features Demonstrated**

- 🍩 Doughnut chart showing recyclable vs. non-recyclable waste
- 📈 Line chart displaying daily scan activity
- 📊 Bar chart showing waste category distribution
- 📌 KPI summary cards including:
  - Total Items Scanned
  - Recyclable Items
  - Recycling Success Rate
  - Hazardous Waste Detections
